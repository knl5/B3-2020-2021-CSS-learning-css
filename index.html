Learning CSS

Sur les troisdernières années CSS s’est dotée de nouveautés et d'innovations qui sont à la foisexcitantes et assoient la CSS comme une discipline incontournable pour tout professionneldu Web. Dans cet écosystème hyper dynamique et changeant, l’important est decomprendre les principes fondamentaux et de savoir quand et comment adopter cesnouveautés. C’est aussi de s’amuser.
