Learning CSS

L'objectif des sessions de travail LEARNING CSS est de permettre aux étudiants d'instaurerune autonomie dans l’acquisition des techniques et des nouveautés en CSS. Sur les troisdernières années CSS s’est dotée de nouveautés et d'innovations qui sont à la fois excitantes et assoient la CSS comme une discipline incontournable pour tout professionneldu Web. Dans cet écosystème hyper dynamique et changeant, l’important est decomprendre les principes fondamentaux et de savoir quand et comment adopter ces nouveautés. C’est aussi de s’amuser.
