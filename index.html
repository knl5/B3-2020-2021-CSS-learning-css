Bienvenue sur Learning CSS

L'objectif des sessions de travail LEARNING CSS est de permettre aux étudiants d'instaurerune autonomie dans l’acquisition des techniques et des nouveautés en CSS. Sur les troisdernières années CSS s’est dotée de nouveautés et d'innovations qui sont à la fois excitantes et assoient la CSS comme une discipline incontournable pour tout professionneldu Web. Dans cet écosystème hyper dynamique et changeant, l’important est decomprendre les principes fondamentaux et de savoir quand et comment adopter ces nouveautés. C’est aussi de s’amuser.

Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) permet de créer des pages web à l'apparence soignée. Il s'agit d'un langage de style dont la syntaxe est extrêmement simple mais son rendement est remarquable. En effet, le CSS s’intéresse à la mise en forme du contenu intégré avec du HTML.

Petite histoire du CSS

Comme pour HTML, CSS a commencé petit et grandit au fil des versions. La première version a vu le jour vers 1996. Il s'agissait de CSS1, suivie de CSS2 qui était le plus populaires jusqu'à l’apparition de HTML5 qui a intégré de nouvelles fonctionnalités qui ont emmené à développer la version 3 de CSS (dite CSS3) qui est le standard le plus utilisé actuellement. On utilise même le terme HTML5/CSS3 pour désigner les deux technologies qui vont alors ensemble

CSS peut être utilisé pour une mise en forme élémentaire des documents — par exemple changer la couleur et la taille des titres et des liens. Il peut être utilisé pour concevoir une maquette — par exemple transformer un texte affiché sur une colonne en une composition avec un cadre principal et une barre latérale pour les informations reliées. Avec CSS, on peut aussi produire des animations. CSS est un langage basé sur des règles — on définit des règles de styles destinées à des éléments ou des groupes d'éléments particuliers dans la page.
